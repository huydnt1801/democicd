clone project:\
`git clone git@github.com:huydnt1801/democicd.git`\
mở code bằng VS Code:\
`code democicd`\
build docker image:\
`docker build -t tên:v1 .`\
xem các image:\
`docker images`\
chạy image:\
`docker run -p port_local:port_container id/tên_image`\
xem các container đang chạy:\
`docker ps`
